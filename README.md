Excel 自动化处理工具（GUI 桌面版）
基于 Python + Pandas + Tkinter + Openpyxl 开发的可视化桌面端 Excel 自动化工具，提供图形化操作界面，无需命令行，一键完成多文件合并、数据清洗、缺失值处理、透视表生成、数据统计、重复值检测等全流程数据处理，操作简单、功能全面，适合简历项目展示与实际办公使用。

核心功能
1. 可视化 GUI 界面：Tkinter 图形窗口，按钮操作、路径选择、进度提示，小白友好
2. 多文件批量合并：自动读取文件夹内所有 Excel/CSV，一键合并汇总
3. 数据清洗：缺失值填充 / 删除、格式统一、异常值过滤
4. 智能去重 / 查重：单列 / 多列查重、重复值标记、自动去重
5. 透视表生成：自定义行 / 列 / 值，自动生成汇总透视表
6. 数据统计分析：均值、总和、最值、计数、占比、分布统计
7. 格式美化导出：自动调整列宽、表头加粗、多 Sheet 规范导出

技术栈
1.开发语言：Python
2.GUI 框架：Tkinter
3.数据处理：Pandas、Openpyxl、Xlrd
4.适用场景：桌面端办公自动化、数据处理、报表生成

使用说明
1.安装依赖：pip install -r requirements.txt
2.运行主程序：python main.py
3.在图形界面选择文件夹 → 点击对应功能按钮 → 自动生成结果文件
4.支持合并表、清洗表、透视表、统计表批量导出
